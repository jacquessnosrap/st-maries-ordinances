# About

This is an mdBook project that contains the ordinances of the Village/City of Saint Maries, Idaho. The ordinances are organized into sections based on their ordinance numbers, and each ordinance is presented in its own markdown file. The project also includes an introduction and a summary of the ordinances at a glance.

This is an ongoing project and its contents are not considered authoritative in any way.


# Setup

Install mdbook:


```bash
    cargo install --git https://github.com/rust-lang/mdBook.git mdbook
```


# Serving the Book

```bash
    mdbook serve --open
```
